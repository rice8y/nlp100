00. 文字列の逆順#

文字列"stressed"の文字を逆に (末尾から先頭に向かって) 並べた文字列を得よ.

text = "stressed"
rev_text = text[::-1]
print(rev_text)

desserts

01. 「パタトクカシーー」#

「パタトクカシーー」という文字列の1, 3, 5, 7文字目を取り出して連結した文字列を得よ.

text = "パタトクカシーー"
sel_text = "".join([text[i] for i in range(0, len(text), 2)])
print(sel_text)

パトカー

02. 「パトカー」+「タクシー」＝「パタトクカシーー」#

「パトカー」+「タクシー」の文字を先頭から交互に連結して文字列「パタトクカシーー」を得よ.

text1 = "パトカー"
text2 = "タクシー"
con_text = []
con_text = "".join([(text1[i] + text2[i]) for i in range(0, len(text1))])
print(con_text)

パタトクカシーー

03. 円周率#

"Now I need a drink, alcoholic of course, after the heavy lectures involving quantum mechanics."という文を単語に分解し, 各単語の (アルファベットの) 文字数を先頭から出現順に並べたリストを作成せよ.

text = "Now I need a drink, alcoholic of course, after the heavy lectures involving quantum mechanics."
words = text.replace(",", "").replace(".", "").split()
word_lengths = [len(word) for word in words]
print(word_lengths)

04. 元素記号#

"Hi He Lied Because Boron Could Not Oxidize Fluorine. New Nations Might Also Sign Peace Security Clause. Arthur King Can."という文を単語に分解し, 1, 5, 6, 7, 8, 9, 15, 16, 19番目の単語は先頭の1文字, それ以外の単語は先頭の2文字を取り出し, 取り出した文字列から単語の位置 (先頭から何番目の単語か) への連想配列 (辞書型もしくはマップ型) を作成せよ.

text = "Hi He Lied Because Boron Could Not Oxidize Fluorine. New Nations Might Also Sign Peace Security Clause. Arthur King Can."
words = text.replace(".", "").split()
tgt_pos = {1, 5, 6, 7, 8, 9, 15, 16, 19}
elem_sym = {i + 1: (word[0] if i + 1 in tgt_pos else word[:2]) for i, word in enumerate(words)}
print(elem_sym)

05. n-gram#

与えられたシーケンス (文字列やリストなど) からn-gramを作る関数を作成せよ. この関数を用い, "I am an NLPer"という文から単語bi-gram, 文字bi-gramを得よ.

def n_gram(tgt, n):
    return [tgt[i:i+n] for i in range(len(tgt) - n + 1)]

text = "I am an NLPer"
words = text.split(" ")
print(n_gram(text, 2))
print(n_gram(words, 2))

['I ', ' a', 'am', 'm ', ' a', 'an', 'n ', ' N', 'NL', 'LP', 'Pe', 'er']
[['I', 'am'], ['am', 'an'], ['an', 'NLPer']]

06. 集合#

"paraparaparadise"と"paragraph"に含まれる文字bi-gramの集合を, それぞれ, XとYとして求め, XとYの和集合, 積集合, 差集合を求めよ. さらに, 'se'というbi-gramがXおよびYに含まれるかどうかを調べよ.

textA = "paraparaparadise"
textB = "paragraph"
textA = n_gram(textA, 2)
textB = n_gram(textB, 2)

X = set(textA)
Y = set(textB)

print(f"Union: {X | Y}")
print(f"Intersection Set: {X & Y}")
print(f"Difference Set: {X - Y}")

tgt = "se"
if tgt in X and tgt in Y:
    print(f"{tgt} exists in X and Y.")
elif tgt in X:
    print(f"{tgt} exists in X.")
elif tgt in Y:
    print(f"{tgt} exists in Y.")
else:
    print(f"{tgt} does not exists.")

Intersection Set: {'ar', 'ap', 'pa', 'ra'}
Difference Set: {'di', 'se', 'ad', 'is'}
se exists in X.

引数x, y, zを受け取り「x時のyはz」という文字列を返す関数を実装せよ. さらに, x=12, y="気温", z=22.4として, 実行結果を確認せよ.

def print_temp(x, y, z):
    print(f"{x}時の{y}は{z}")

08. 暗号文#

与えられた文字列の各文字を, 以下の仕様で変換する関数cipherを実装せよ.

  • 英小文字ならば(219 - 文字コード)の文字に置換

  • その他の文字はそのまま出力

この関数を用い, 英語のメッセージを暗号化・復号化せよ.

def cipher(text):
    res = []
    for char in text:
        if 'a' <= char <= 'z':
            new_char = chr(219 - ord(char))
            res.append(new_char)
        else:
            res.append(char)
    return ''.join(res)

text = "Hello, World!"
res = cipher(text)
print(res)

Hvool, Wliow!

09. Typoglycemia#

スペースで区切られた単語列に対して, 各単語の先頭と末尾の文字は残し, それ以外の文字の順序をランダムに並び替えるプログラムを作成せよ. ただし, 長さが4以下の単語は並び替えないこととする. 適当な英語の文 (例えば"I couldn't believe that I could actually understand what I was reading : the phenomenal power of the human mind .") を与え, その実行結果を確認せよ.

import random

def typoglycemia(text):
    words = text.split()
    for i in range(len(words)):
        if len(words[i]) <= 4:
            continue
        else:
            word = words[i]
            middle = word[1:-1]
            shuffled_middle = ''.join(random.sample(middle, len(middle)))
            words[i] = word[0] + shuffled_middle + word[-1]

    return ' '.join(words)

text = "I couldn't believe that I could actually understand what I was reading : the phenomenal power of the human mind ."
res = typoglycemia(text)
print(res)
